Domain Recon Tool



A Python command-line tool that collects basic reconnaissance information about a domain. It gathers DNS records, WHOIS information, SSL certificate details, open ports, and common subdomains, and saves the results as a report.



Features:

* Resolve domain IP address
* Retrieve DNS records 
* Perform WHOIS lookup
* Check SSL certificate information
* Scan common ports
* Discover common subdomains
* Generate a simple infrastructure graph
* Save results as a JSON report





Usage:



1. Run the script and provide a domain:



&nbsp;	python recon.py example.com





2\. You can also pass a URL:

&nbsp;	python recon.py https://example.com
